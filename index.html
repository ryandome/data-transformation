Data Transformation

A1 

    var getGrade = function(quiz)
 {
    return quiz.grade;
 }

This function returns the values for the grade attribute.

A2 

    d3.mean(classroom[0].quizes, getGrade)

This calculates the mean of the quiz grades for th first penguin. It uses the function getGrade, and specifies that it should pull the grades from the objects in the quizes array for the first penguin.

A3

    var Mean =function(penguin)
{
    return d3.mean(penguin.quizes, getGrade)
}

This function returns the average quiz grade for a given penguin. It combines A1 and A2 to yeild a genrealized mean function.

A4 

    classroom.map(Mean)

This calls the mean function for every element (penguin) in the classroom array and puts it into a new array.

A5

    d3.mean(classroom.map(Mean))

This calls all the values from the array created in step A4. It then calculates the mean of all of those values.
